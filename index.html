How to Get Your API Key

  1. Sign up for a free account at OpenWeatherMap
  2. Go to your API keys section
  3. Copy your API key
  4. Open config.js and replace "YOUR_API_KEY_HERE" with your actual API key
  5. Important: Never commit your config.js file to a public repository
